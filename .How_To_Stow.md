# Naming Convention

**dotfiles              ->    GNU Stow**
~/.config/nvim        ->    nvim/.config/nvim
~/.config/aerospace   ->    nvim/.config/aerospace
~/.zshrc              ->    zsh/.zshrc

# Steps
1. `mv ~/.config/nvim ~/.dotfiles/nvim/.config`
2. `stow nvim`:: symlinked `.dotfiles/nvim/.config/` to `.config/nvim` 

# Notes
  + `.config/nvim` must be the one pointing to `.dotfiles/nvim`
  + do not need to commit plugins to git. The plugin's manager will install the plugins from the config file
    + EX: `.gitignore`, inside: `tmux/.config/tmux/plugins/
